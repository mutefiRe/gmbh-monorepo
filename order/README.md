# GmBh Frontend

####Setup
#####General
```
npm install -g babel-eslint
npm install -g eslint
npm install -g commitizen
```
[Install GIT-Flow](https://github.com/nvie/gitflow/wiki/Installation)
#####Frontend
```
npm install -g eslint-config-ember
npm install -g ember-cli
npm install -g bower
```
#####Install
```
npm install
bower install
```
