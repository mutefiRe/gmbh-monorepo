Zertifikat installieren

Um das lokale Bestellsystem sicher zu nutzen, bitte das folgende Zertifikat installieren:

1. Zertifikat herunterladen

myCA.pem herunterladen

Falls du ein Tablet nutzt: Tippe einfach auf den Link.

2. Installation

Android

  1. Benachrichtigung öffnen: "Zertifikat installieren"
  2. "VPN & Apps" oder "WLAN" auswählen
  3. Bestätigen

iOS

  1. Zertifikat laden
  2. Einstellungen → Profil
  3. Profil installieren
  4. Einstellungen → Allgemein → Info → Zertifikatvertrauenseinstellungen → Zertifikat aktivieren
